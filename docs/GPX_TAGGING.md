# GPX Tagging

Startpunkt:

```text
#type:start
#ort:Stuttgart
#date:01.07.2026
#starttime:11:45
```

Overnight-Punkte:

```text
#type:overnight
#ort:Bebenhausen
#starttime:09:00
```

Zielpunkt kann ADFC-/Tourlinks enthalten:

```text
#adfcStars:4
#adfcUrl:www.adfc-radtourismus.de/hohenzollern-radweg/
#officialUrl:www.schwaebischealb.de/rad/hohenzollern-radweg
```
