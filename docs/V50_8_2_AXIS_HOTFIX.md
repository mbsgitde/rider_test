# V50.8.2 Axis Hotfix

Gezielter Hotfix für die Gesamtübersicht:

- Overnight-KM-Labels werden auf Höhe der X-Achsen-Ticklabels gezeichnet, also auf Höhe der normalen `0`.
- Overnight-Vertikallinien sind durchgezogen und dezent im Stil der Koordinatensystemlinien.
- Die X-Achse der Gesamtübersicht erhält explizite Boundary-Ticks: `0`, alle Etappen-/Overnight-Grenzen und die gerundete Gesamtstrecke, z. B. `260`.

Anwendung: Diese ZIP auf eine funktionierende V50.8.1-Version entpacken und `assets/app.js` ersetzen.
