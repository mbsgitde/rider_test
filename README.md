# Bike Tour Planner – GitHub Pages Starter

Dieses ZIP enthält ein vollständiges Starter-Repo für GitHub Pages auf Basis des finalen PRD.

## Inhalt

```text
index.html
assets/
  app.js
  styles.css
gpx/
  alps.gpx
  lake.gpx
data/
  config.json
  config.md
  alps-stops.json
  lake-stops.json
PRD_Bike_Tour_Planner_Best_Practice.md
```

## Enthaltene Funktionen
- Tourauswahl per Dropdown
- Laden externer GPX-Dateien
- Laden externer Stop-JSON-Dateien
- Laden zentraler `config.json`
- Distanz, Höhenmeter, Netto-/Brutto-Fahrzeit, Schwierigkeit
- Höhenprofil pro Etappe
- Unterkunftsanzeige mit Link
- Plausibilitätswarnung für Stops neben der Route
- farbige Etappen auf der Karte
- linker farbiger Vertikalmarker pro Etappe in der Übersicht

## Deployment auf GitHub Pages
1. ZIP entpacken
2. Inhalte in ein GitHub-Repository pushen
3. GitHub Pages auf Branch `main` und Root aktivieren
4. Seite öffnen
